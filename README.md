# FreePDF Pro

Free online PDF tools to edit, convert, compress, and merge PDF files. 100% free, secure, and runs entirely in your browser.

## Features

- 📑 Merge PDF - Combine multiple PDFs
- ✂️ Split PDF - Extract specific pages
- 🗜️ Compress PDF - Reduce file size
- 🖼️ PDF to JPG/PNG - Convert to images
- 📷 Image to PDF - Create PDF from images
- 🔄 Rotate PDF - Fix orientation
- 🔒 Protect PDF - Add password encryption
- 🔓 Unlock PDF - Remove passwords

## Tech Stack

- Frontend: HTML5, CSS3, Vanilla JavaScript
- PDF Processing: PDF-lib, PDF.js (client-side)
- Backend: Node.js, Express
- Hosting: Render.com

## Deployment

This app is deployed on Render.com with automatic deployments from GitHub.
